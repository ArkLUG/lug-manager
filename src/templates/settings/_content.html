Member Import

Pull all members from the Discord server and automatically create accounts for anyone whose Discord roles map to a LUG role. Existing members get their display name and role updated. Dues and paid status are never overwritten. The sync also runs automatically every 6 hours in the background.

Note: The Discord bot token is set via the DISCORD_BOT_TOKEN environment variable and cannot be changed here. Each chapter has its own announcements channel configured on the chapter's settings page.
